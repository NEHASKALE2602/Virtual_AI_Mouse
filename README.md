# 🖱️ NeuroMouse AI

NeuroMouse AI is an AI-powered desktop application that allows users to control their computer using real-time hand gestures captured through a webcam. The project uses Computer Vision and Artificial Intelligence to provide a touch-free virtual mouse experience.

---

## 📌 Features

- ✅ AI-powered Hand Tracking
- ✅ Smooth Mouse Movement
- ✅ Left Click
- ✅ Scroll Control
- ✅ Drag & Drop
- ✅ Real-time Gesture Recognition
- ✅ Live Camera Feed
- ✅ FPS Monitoring
- ✅ AI Confidence Display
- ✅ Modern Dark Dashboard UI
- ✅ Gesture Guide
- ✅ Settings Page
- ✅ About Page

---

## 🚀 Installation

Clone the repository:

```bash
git clone <repository-link>
```

Open the project folder:

```bash
cd Virtual_AI_Mouse
```

Install the required packages:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python main.py
```

## 🎯 Project Objective

The objective of NeuroMouse AI is to demonstrate how Artificial Intelligence and Computer Vision can be used to create a touch-free desktop interaction system by replacing traditional mouse operations with natural hand gestures.
